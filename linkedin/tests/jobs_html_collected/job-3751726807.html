      Que desafio temos para você?


      Estamos em busca de um(a) Engenheiro(a) de Dados Sênior para atuar no nosso time de Business Intelligence da área Comercial. Nosso objetivo é suportar a tomada de decisões através do gerenciamento de produtos de dados (KPIs, pipelines, relatórios, análises) e da geração de insights.


      Quais as responsabilidades?


      • Gerir ferramentas de orquestração, integração e transformação de dados;
      • Suportar e melhorar nosso Data Warehouse e outros produtos de dados;
      • Trabalhar em estreita colaboração com os analistas de dados na criação de novos produtos;
      • Definir e seguir boas práticas em desenvolvimento de software, arquitetura de dados e documentação;
      • Realizar projetos de integração de dados com times internos e consultorias externas;
      • Resolver incidentes e problemas técnicos;
      • Contribuir com a gestão da rotina e capacitação do time;
      • Fomentar uma cultura de dados dentro e fora da equipe.


      Quais os requisitos?

      • Ensino superior completo;
      • Inglês avançado;

      Conhecimento e experiência em:

      • Engenharia de software, algoritmos e estruturas de dados;
      • ETL/ELT (preferencialmente com SQL e Python);
      • Modelagem de dados multidimensionais (ex: star schema);
      • Soluções para armazenamento e gestão de dados analíticos (ex: Data Warehouse, Data Lake);
      • Ferramentas para orquestração de pipelines de dados (ex: Airflow, Dagster, Mage);
      • Ferramentas para integração de dados (ex: Talend, Airbyte, Apache NiFi);
      • Utilização de APIs para integração e consumo de dados;
      • Serviços de armazenamento e computação em nuvem (ex: AWS S3, AWS EC2, AWS Redshift);
      • Versionamento de código com Git;
      • CI/CD;
      • Linux;
