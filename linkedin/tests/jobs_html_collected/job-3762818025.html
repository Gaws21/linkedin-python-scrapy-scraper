      Job Description

      • Capturar dados nos sistemas legados via API;

      • Garantir as integridades das API’s existentes, desenvolver plataformas completas do frontend e backend (com bancos de dados, servidores e clientes);
      • Levantar requisitos desde regras de negócio, necessidades de infraestrutura e necessidade dos usuário;
      • Realizar o versionamento de código com Git e Gitlab;
      • Desenvolver API REST (backend);
      • Configurar servidores utilizando shell linux;
      • Criar e configurar o banco de dados SQL e NoSQL. Modelagem e estruturação de dados (tabelas, schemas e collections);
      • Noções de ciência de dados é desejável (modelos de previsão, classificação e clusterização, mecanismos de associação, estudos).

      Requirements

      • Inglês intermediário;
      • Conhecimento em Python e Javascript;
